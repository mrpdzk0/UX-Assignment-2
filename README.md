My aims and objectives for this website were:
- to make an easy to use movie website for the entertainment of people, it has an easy to see list of movies and more to find the right one for you and dark mode to make it easier on users who are sensitive to bright light

I implemented a dark background and large contrasting colours for our first profile "Ruben Washington" as he has visual impairments and his preference are dark backgrounds. Contrasting titles make them easier to see even with visual impairments like him so he should have no trouble at all using this website.
